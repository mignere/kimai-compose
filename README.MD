## ini repo cuma buat nyimpen compose kimai sekalian biasain make git

### how to 
1. clone repo
    * \# git clone https://github.com/mignere/kimai-compose.git
2. jalanin compose
    * \# docker compose up 
3. add user admin
    * \# /opt/kimai/bin/console kimai:user:create admin admin@example.com ROLE_SUPER_ADMIN
